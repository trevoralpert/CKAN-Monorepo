Docs and Download

Documentation

Full developer documentation covering:

  • Installation and configuration
  • Customization and authorization
  • Extensions, various features and more

Download

Binaries

Built binaries for Ubuntu and Debian can be found at: http://apt.ckan.org/.

Detailed instructions on a package installation including support Ubuntu and Debian versions can be found in the CKAN Package install section of the documentation.

Source Code

The CKAN source code is available from this public repository in github: https://github.com/ckan/ckan.

Details of the source install are in the “Install from Source” section of the CKAN documentation.
