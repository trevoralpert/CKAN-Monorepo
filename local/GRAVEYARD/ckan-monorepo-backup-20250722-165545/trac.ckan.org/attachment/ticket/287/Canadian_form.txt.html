302. Proposed Format for 'Register a  New Package' Form
31--------------------------------------------------------------------------------
32
33In this section, [square brackets] indicate technical notes/questions. Otherwise, all text should be displayed on the form (including instructions above/before fields and examples below/after).
34
35
36Short Unique Name (required)
37Insert a short, descriptive title here, using only 'a-z', '0-9' and '-_'. Must be lowercase; no spaces allowed.
38______________________________
39(Example: geogratis-radarsat-mosaic)
40
41Title
42Insert a more descriptive title - ideally the same as that used on the government website.
43_______________________________
44(Example: RADARSAT Ortho-rectified Mosaic of Canada, Lambert Conformal Conic, 1000 Metres)
45
46URL
47[Is it possible to add a default  'http://' to the URL & Resources  fields? Seems like it might reduce  error & make data entry a bit  quicker to boot.]
48The link to the government web page where the data is described and located. Do not link to the dataset itself.
49http://_________________________
50(Example: http://geogratis.cgdi.gc.ca/geogratis/en/collection/detail.do?id=9369)   
51
52Resources
53The link to the actual data set (if available)
54http://_________________________
55(Example: http://geogratis.cgdi.gc.ca/geogratis/en/collection/metadata.do;jsessionid=C7EC7276E2CC15D6A5201FA98F19406D?id=9369)
56
57 Format
58Describe the format type, e.g. XML, CSV, SHAPE, etc.
59_______________
60
61 Description
62 A description of the data. Feel free to copy the description on the government website.
63
64 Hash
65For über data geeks - if you don't know what this means, just leave it blank.
66
67
68Date Released
69Please enter in YYYY-MM-DD format.
70______________________________
71(Example: 2009-01-15)
72
73Date Updated
74Please enter in YYYY-MM-DD  format.
75______________________________
76(Example: 2010-04-12)
77
78Update frequency (only if available)
79______________________________
80(Example: monthly)
81
82Years covered [was 'Temporal coverage' in UK form]
83Please enter in YYYY - YYYY format.
84[David R: Can we validate these fields to force them to be entered as YYYY?]
85________ - ________
86(Example: 1992 - 1995)
87
88
89License
90Select the license type.
91[drop-down…  See list at bottom of this document. Note: We will need to add to this over time.]
92
93Notes
94A  more detailed description of the data can go here. Feel free to copy  & paste from the government website.
95___________________________
96
97Tags  (space separated list)     
98Include tags you think are descriptive and appropriate. Tags are space separated; to  join two words together, use a dash (-) or underscore (_).
99___________________________
100(Example:  radar radar-imagery radarsat-1 satellite-imagery mosaic  spectral-engineering geogratis canada gcmd earth-science)
101
102
103Level of Government [was 'Geographic granularity' in UK form]
104What level of government produced this data?
105[Pull-down menu with the following options: Federal, Provincial/Territorial, Regional, Municipal, Other - please specify]
106____ [pull-down] _____  or Other: ___ [text field] ___
107
108Federal Ministry (if applicable) [was 'Department' in UK form]
109Indicate the ministry responsible for the data.
110 [pull-down + other field]
111
112Federal Agency [was 'Agency' in UK form]
113Indicate the agency responsible for the data.
114[Can we make this a pull-down menu + other field? List appended below.]
115
116Maintainer email
117Include an email address for the ministry or agency responsible for maintaining the data (if available).
